Working with CSS3 Layout

The viewport meta tag is located at the head of a web page. This meta tag is going to tell the browser how to show a web page on different devices, the most important is defining how the HTML webpage should scale and fit on any screen, from a desktop to a small phone.

Working with images

The img tag is going to use 2 required attributes

  • Source tag= src, is going to be used to place the path where the image is.
  • Alternate tag= alt, is used for SEO, to describe the image and is also is going to show a text in the browser if the image can't be found
  • Title attribute, we are going to create a specific sentence using the main keyword if the page
